# BGMファイルの配置

このフォルダに、FMラジオっぽいBGMファイルを配置してください。

## 推奨される音源

- **ファイル名**: `fm-radio-bgm.mp3` または `fm-radio-bgm.ogg`
- **形式**: MP3 または OGG
- **特徴**: 
  - FMラジオ番組のBGMのような雰囲気
  - ループ再生可能な長さ（30秒〜2分程度）
  - 薄く流れる程度の音量で使用されるため、控えめな音源
  - 会話の邪魔にならない、環境音的なBGM

## フリー音源の推奨サイト

- [Freesound](https://freesound.org/) - フリー音源の検索
- [YouTube Audio Library](https://www.youtube.com/audiolibrary) - ロイヤリティフリー音源
- [Incompetech](https://incompetech.com/music/royalty-free/) - ロイヤリティフリー音楽
- [Bensound](https://www.bensound.com/) - フリー音楽

## 検索キーワード

- "ambient background music"
- "radio background music"
- "soft background music"
- "looping background music"
- "minimal background music"

## 注意事項

- 音源ファイルは必ずロイヤリティフリーのものを使用してください
- 商用利用可能なライセンスを確認してください
- ファイルサイズは可能な限り小さく（1MB以下推奨）
